Package xdi2.messaging.target.interceptor

Interface Summary
Interceptor  
MessageEnvelopeInterceptor Interceptor that is executed before and after a message envelope is executed, as well as when an exception occurs while executing the message envelope.
MessageInterceptor Interceptor that is executed before and after a message is executed.
MessagingTargetInterceptor Interceptor that is executed when a messaging target is initialized and shut down.
OperationInterceptor Interceptor that is executed before and after an operation is executed.
ResultInterceptor Interceptor that is executed on a message result after a message envelope has been executed.
TargetInterceptor Interceptor that is executed when an operation is executed on an XDI statement or an XDI address.
