xdi2.core.features.multiplicity
Class Multiplicity

java.lang.Object
  xdi2.core.features.multiplicity.Multiplicity

public class Multiplicity
extends Object

Multiplicity supports constructs to express XDI collections, entities and attributes. All these constructs are context nodes whose arc XRI follow certain rules. Examples: Collection: $(+tel), $(+passport) Entity Singleton: =!1111, +passport Attribute Singleton: $!(+tel) Entity Member: $(!1) Attribute Member: $!(!1)
